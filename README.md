# 项目概况
本项目提供完全免费的JAVA版Discuz Ucenter API，可以轻松实现现有JAVA系统与UCenter之间无缝对接。具体实现在的功能如下：
1.单点登录, Discuz! passport for java.
2.基本用户管理的API。
